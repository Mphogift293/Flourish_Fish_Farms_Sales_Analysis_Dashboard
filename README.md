# Flourish_Fish_Farms_Sales_Analysis_Dashboard 2026

### About this Project
Dashboard I built to track sales, costs & profit for a fish farming business across 5 states in Nigeria using Google Sheets.

### Key Results
-**Total Revenue**: $165,195
-**Total Costs**: $65,854
-**Total Profit**: $99,341
-**Profit Margin**:60.14%
-**Top State**: Abuja- $29,254 profit
**Top Product**: Bullhead- $79,860 revenue

### Charts Included
1. Donut: Profit per State
2. Line: Revenue vs Profit vs Cost by month
3. Bar: Profit per Product per State
4. Donut: Revenue per Product

 ### Tools & Skills
 Google Sheets, SUMIFS Formulas, Pivot Tables, Charts
